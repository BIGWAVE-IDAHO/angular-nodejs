This document describes the URL search API that the Cardetektive webapp provides.

The API is quite simple. Basically it expects the normal Carsearch URL, which is {{ clientBaseUrl }}car-search, populated with some of the supported search parameters as URl parameters.

The "vin-search-mileage" should be a valid number. The zip-code needs to be a valid zip-code. The only parameter with no restrictions is the "version".' | translate }}
